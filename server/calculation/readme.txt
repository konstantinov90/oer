1. Поставить Python 3
2. Поставить python пакет pyomo.
3. Прописать в последней строчке mian.py каталоги, в которых находятся модель и заявки. Примеры файлов модели и заявок приложены. Файлы моделей уже описывают объекты общего рынка и менять их не следует. В имитационных торгах меняются только заявки. Из директории с заявками считываются все файлы.
4. Запустить python main.py (в linux python3 main.py).
5. Результаты расчета пока выводятся только на экран.
